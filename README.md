## gist

`go get -u -v bitbucket.org/jsxqf/gist-cli/cmd/gist`

```sh
Usage of gist:
  -d string
    	description
  -p	make gist public [false] or secret [true]
  -py string
    	(socks5, http, https) proxy
  -t string
    	token for gist (https://github.com/settings/tokens)
  -u string
    	username
```
